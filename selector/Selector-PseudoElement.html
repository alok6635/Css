Pseudo-element

CSS pseudo-element is used to style specified parts of an element

  • selection
  • marker
  • first-line
  • first-letter
  • after
  • before

Selection-

selection pseudo-element matches the portion of an element that is selected by a user.

Selection Ex-

select content to the mouse then change color

Marker-

marker pseudo-element selects the markers of list items.

Marker Ex-

  1. one item
  2. two item
  3. three item

First letter-

First letter Ex-

You can combine the ::first-letter and ::first-line pseudo-elements to add a special effect to the first letter and the first line of a text!

Note - 1. first-line,first letter pseudo-element can only be applied to block-level elements.

::before,after Pseudo-element,-

Ex-before,after Pseudo-element-

::after pseudo-element can be used to insert some content after the content of an element.

The ::before pseudo-element inserts content before the content of an element.
